# Noteful Client

<p> This app allows the user to create and delete folders. User can create new notes and organize them into specific folders as well as delete them.
</p>

### Client-side used:
<ul>
  <li>ReactJS</li>
  <li>Context</li>
</ul>

### Server-side used:
<ul>
  <li>NodeJS</li>
  <li>Express</li>
  <li>Knex</li>
</ul>
